Measure differences between 2 identically sized buffer arrays of gray, rgb, or rgba pixels.

## installation
```
npm install pixel-change
```
## testing
```
git clone https://github.com/kevinGodell/pixel-change.git
cd pixel-change
npm install
npm test
```
